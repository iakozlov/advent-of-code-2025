# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Go.

## Running Solutions

Navigate to the day's directory and run:

```bash
cd day1
go run . 1    # Run Part 1
go run . 2    # Run Part 2
```
