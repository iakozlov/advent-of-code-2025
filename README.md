# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Go.

## Running Solutions
```bash
go run ./cmd/day1 1    # Day 1 Part 1

go run ./cmd/day2 2    # Day 2 Part 2
```

## Adding Input Files

Create an `input/` folder inside each day directory and add the puzzle input:

```
day1/input/input.txt
day2/input/input.txt
```
